# di-fugen

Website für **di-fugen** – Fugenarbeiten, Silikonfugen und Fugensanierung für
Bad, Küche und Fenster. Spezialisiert auf elastische Fugen und PU-Fugen.

## Inhalt

| Datei              | Seite                          |
| ------------------ | ------------------------------ |
| `index.html`       | Startseite                     |
| `leistungen.html`  | Leistungen                     |
| `ueber-uns.html`   | Über uns                       |
| `referenzen.html`  | Referenzen                     |
| `kontakt.html`     | Kontakt                        |
| `impressum.html`   | Impressum (Mustervorlage)      |
| `datenschutz.html` | Datenschutz (Mustervorlage)    |
| `assets/style.css` | Gemeinsames Stylesheet         |

## Lokale Vorschau

Statische HTML-Seiten – einfach `index.html` im Browser öffnen oder einen
lokalen Server starten:

```bash
python3 -m http.server 8000
# danach http://localhost:8000 aufrufen
```

## Hinweise

Texte, Bilder, Kontaktdaten sowie Impressum und Datenschutz sind Platzhalter
und sollten vor dem Live-Gang an das eigene Unternehmen angepasst werden.
